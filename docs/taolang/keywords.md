# 关键字参考

本文档列出 TaoLang 中所有关键字的分类与说明，包括已定义关键字、设计中关键字以及保留关键字。

---

> 注意：TaoLang 编译器尚处于设计阶段，以下所有关键字均未在编译器中实现。本文档记录的是语言规范中的设计方案。

## 已定义关键字

以下关键字已纳入语言规范，语义明确：

### 控制流

| 关键字 | 说明 |
|--------|------|
| `if` | 条件分支。根据表达式的布尔值选择执行路径。 |
| `else` | 条件分支的备选路径。当 `if` 条件不满足时执行。 |
| `while` | 条件循环。在条件为真时重复执行循环体。 |
| `for` | 迭代循环。使用 `for x in collection` 语法遍历可迭代对象中的每一个元素。 |
| `do` | 配合 `loop` 使用的循环结构，定义循环体的开始。 |
| `loop` | 循环控制关键字。支持次数和间隔参数，如 `loop(100, sep=1000)` 表示循环 100 次、每次间隔 1000 毫秒。 |
| `break` | 跳出当前循环，或跳出当前生命周期空间。 |
| `continue` | 跳过当前循环迭代的剩余部分，直接进入下一次迭代。 |
| `return` | 从函数中返回值，并结束函数执行。 |
| `pass` | 空操作占位符。不执行任何操作，不产生任何效果。用于语法上需要语句但逻辑上不需要执行任何操作的场景。例如：`when modify { pass }` 等价于 `when modify {}`，`pass` 仅用于增强代码可读性。 |
| `del` | 主动删除变量，触发 delete 和 free 生命周期事件（对应的 `when`/`on` 钩子均会执行）。 |

**示例：**

```tao
fn countdown(n: int) -> int {
    let i = n
    while i > 0 {
        if i == 5 {
            println("halfway!")
        }
        i = i - 1
    }
    return 0
}
```

```tao
// loop 示例：循环 10 次，每次间隔 500 毫秒
do loop(10, sep=500) {
    println("tick")
}
```

### 变量与常量

| 关键字 | 说明 |
|--------|------|
| `let` | 定义局部可变变量。变量作用域限定在当前块内。 |
| `const` | 定义编译期常量。全局可见，值在编译时确定，不可修改。 |
| `def` | 定义全局变量。声明时必须赋值。可在任意位置定义，效果等同于在 null 层级（最顶层作用域）定义。 |
| `as` | 变量委托。用于创建动态变量，每次访问时重新对表达式求值，而非存储固定值。 |

**示例：**

```tao
let count = 0          // 局部可变变量
const MAX = 100        // 编译期常量
def app_name = "demo"  // 全局变量

// 动态变量：每次访问 now 时都会重新调用 get_time()
let now as get_time()
```

### 生命周期与空间

| 关键字 | 说明 |
|--------|------|
| `space` | 声明一个生命周期空间。空间是 TaoLang 中管理作用域与状态的核心单元。 |
| `using` | 进入一个已声明的生命周期空间，激活其内部状态与钩子。 |
| `global` | 引用所有生命周期空间的总体，即全局作用域。 |
| `set` | 通用变量属性设置关键字。支持两种子语法：`set...to` 设置属性值，`set...[self] ->` 追加钩子绑定。`set a on use {...}` 是 `set a [self] -> on use {...}` 的语法糖。 |
| `to` | 与 `set` 配合，指定属性的目标值。 |
| `on` | 被动事件钩子。用于监听生命周期事件，在事件发生后执行回调逻辑。 |
| `when` | 主动事件钩子。用于在事件发生前拦截生命周期事件，可自定义操作逻辑以替代默认行为。 |
| `goto` | 跳转到指定的生命周期空间。 |
| `with` | 表示空间嵌套关系。如 `a with b` 表示空间 a 嵌套在空间 b 中。 |

**示例：**

```tao
space session {
    let user = null
}
    [self] ->
    on enter {
        println("session started")
    }
    on exit {
        println("session ended")
    }

using session {
    user = "alice"
}
```

### 作用域与属性

| 关键字 | 说明 |
|--------|------|
| `in` | 指定变量所在的生命周期空间；在 `for` 循环中用于指定迭代对象。 |
| `of` | 查询变量的从属属性。如 `x of type` 获取类型、`x of space` 获取所属空间。 |
| `self` | 当前绑定实体自身的引用。在钩子、类方法、结构体关联函数和空间函数中需使用 `self.` 前缀访问成员。例外：在 `using` 块中可直接使用变量名，无需 `self.` 前缀。 |
| `super` | 引用父类或父空间，用于访问上层作用域中的成员。 |
| `all` | 表示所有从属成员。 |
| `is` | 类型与身份检查运算符。用于判断值的类型或身份，如 `x is SomeType`。 |

**示例：**

```tao
let x = 42 in session       // 在 session 空间中定义变量
let t = x of type            // 获取变量 x 的类型
let s = x of space           // 获取变量 x 所属的空间
```

### 导入

| 关键字 | 说明 |
|--------|------|
| `import` | 导入模块中的指定项。 |
| `from` | 指定导入来源路径。 |
| `package` | 声明当前文件所属的包。用于在文件头部标识包归属，如 `package std.io`。 |

**示例：**

```tao
from {res/std.io} import {println, readln}
from {res/std.math} import {sqrt, pow}
from {native/models/user} import {User}
from {dep/ui_lib/button} import {Button}
```

### 类型定义

| 关键字 | 说明 |
|--------|------|
| `fn` | 定义函数。支持参数、返回值类型声明与函数体。 |
| `struct` | 定义结构体。轻量级数据容器，用于组织相关数据字段，不可为空。 |
| `class` | 定义类。支持继承、封装以及生命周期钩子绑定。 |
| `override` | 显式覆盖父类的方法或字段。 |

**示例：**

```tao
struct Point {
    x: float,
    y: float
}

class Entity {
    public let name: str
    private let id: int

    fn greet() {
        println($"I am {self.name}")
    }
}

fn distance(a: Point, b: Point) -> float {
    sqrt((a.x - b.x) ** 2 + (a.y - b.y) ** 2)
}
```

### 访问控制

| 关键字 | 说明 |
|--------|------|
| `public` | 公开成员。外部可读可写。 |
| `private` | 私有成员。仅在定义所在的类或空间内部可访问。 |
| `protect` | 受保护成员。外部可读但不可写。 |

**示例：**

```tao
class Account {
    public let name: str        // 外部可读可写
    protect let balance: float  // 外部可读，不可写
    private let secret: str     // 仅内部可访问
}
```

## 统一上下文对象

TaoLang 提供两个内置上下文对象，用于在钩子和空间中访问当前上下文信息：

### `self` -- 当前绑定实体

`self` 始终指向当前绑定的实体自身。在以下上下文中可用：

- **类内部**：指向当前类实例。
- **空间内部**：指向当前生命周期空间。
- **钩子内部**：指向钩子所绑定的对象。

### `it` -- 当前事件上下文参数

`it` 是预定义标识符（不是关键字），仅在部分 `when` 钩子（`create`、`modify`、`enter`）和空间 `on enter` 中由编译器自动生成，携带与当前事件相关的外部输入。其他钩子不传递 `it`，直接通过 `self` 操作。`it` 可通过 `new_val ->` 等语法自定义接收参数名称。

**示例：**

```tao
space counter {
    let value = 0
}
    [self] ->
    on enter {
        // self 指向 counter 空间自身
        println($"entering {self}")
    }

// 变量 on 钩子中通过 self 访问最新状态（不传递 it）
let score = 0
    [self] ->
    on modify {
        println($"score modified to: {self}")
    }
```

### 自定义上下文名称

上下文对象的名称可以自定义。通过 `[自定义名称] ->` 语法可将默认的 `self` 替换为自定义变量名。推荐在大多数场景下使用默认的 `self` 和 `it`，仅在需要区分多层嵌套上下文时使用自定义名称。

```tao
let value = 42
    [ctx] ->
    on modify {
        // ctx 替代 self，指向 value 变量
        println($"modified to: {ctx}")
    }
```

## 保留关键字

以下关键字当前不可使用，已被保留供未来版本扩展。请勿将它们用作标识符。

| 关键字 | 可能用途 |
|--------|----------|
| `where` | 类型约束或条件过滤 |
| `which` | 选择性引用 |

| `except` | 异常捕获 |
| `yield` | 生成器产出值 |

## 内置类型标识符

以下标识符是 TaoLang 的内置类型名称，**不是关键字**，不在关键字速查表中列出。它们由编译器预定义，不可用作变量名：

| 标识符 | 说明 |
|--------|------|
| `int` | 有符号整数。位宽与目标平台指针大小一致（对应 C 的 `intptr_t`、Rust 的 `isize`）。 |
| `float` | 64 位双精度浮点数（对应 C 的 `double`、Rust 的 `f64`）。 |
| `bool` | 布尔值。 |
| `str` | UTF-8 字符串。 |
| `bytes` | 字节数组。 |
| `ptr` | 通用指针（对应 C 的 `void*`）。 |
| `void` | 无返回值类型。用于 `-> void` 标注函数无返回值。 |
| `tuple` | 元组。固定长度的异构集合。 |

> **注意**：`.ti` 文件中的 `link` 和 `type` 也不是关键字，而是接口声明文件的专用指令，仅在 `.ti` 文件中有效。在 `.tao` 文件中，`link` 和 `type` 可用作普通标识符。

## 内置函数

以下函数由编译器预定义，在任何 `.tao` 文件中**无需导入即可直接使用**。它们属于语言基础设施，与 `free`、`del`、`alloc` 等内置操作同等对待：

| 函数 | 签名 | 说明 |
|------|------|------|
| `println` | `(value: any) -> void` | 输出一行文本到标准输出（自动换行）。 |
| `print` | `(value: any) -> void` | 输出文本到标准输出（不换行）。 |
| `readln` | `() -> str` | 从标准输入读取一行文本。 |

> **设计理由**：几乎每个程序都需要基本 I/O 操作。强制导入 `println` 会增加无意义的样板代码，降低入门体验。Python 的 `print`、Rust 的 `println!`、Kotlin 的 `println` 均为内置，TaoLang 遵循这一现代语言惯例。标准库 `res/std.io` 中提供更丰富的 I/O 函数（格式化输出、文件操作等），需显式导入。
